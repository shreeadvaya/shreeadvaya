# ShreeAdvaya - Premium Saree & Dress Materials Collection

A modern, elegant e-commerce website showcasing premium saree collections and unstitched dress materials with WhatsApp integration for business inquiries.

## Features

- **Product Catalog**: Browse sarees and dress materials with collection & price filters
- **Multi-image Products**: Product cards with image carousel navigation
- **WhatsApp Integration**: One-click inquiry with product details
- **Collections System**: Organized by Sarees and Unstitched Salwar Materials with subcategories
- **Admin Panel**: Content management for products, hero images, and website content
- **Responsive Design**: Optimized for desktop, tablet, and mobile

## Collections

- **Sarees**: Kancheepuram Silk, Mysore Silk, Banarasi Silk, Cotton, Linen, Chiffon, Semi Silk, Modal Silk, Silk Cotton
- **Unstitched Salwar Materials**: Linen, Cotton, Chanderi, Modal Silk

## Tech Stack

- HTML5, CSS3, JavaScript
- Vercel Serverless Functions
- GitHub API for content storage
- Font Awesome & Google Fonts

## Deployment

Deployed on Vercel with automatic deploys on push to main branch.

### Environment Variables (Vercel)

- `GITHUB_TOKEN`: GitHub personal access token with Contents read/write
- `ADMIN_PASSWORD`: Admin panel password

## Admin Panel

Access at `/admin` to manage:
- Products (add, edit, delete with multiple images)
- Hero slideshow images
- Website content and contact info
- Collections and subcategories
